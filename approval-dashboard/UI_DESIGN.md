# UI Design Guidelines

## Color Scheme
- Primary: Purple (#7C3AED) to Blue (#3B82F6) gradient
- Success: Green (#10B981)
- Warning: Orange (#F59E0B)
- Danger: Red (#EF4444)
- Neutral: Gray scale

## Typography
- Headers: Bold, large
- Body: Regular, readable
- Code/Quotes: Monospace, gray background

## Components Style

### Issue Card
- White background
- Rounded corners (12px)
- Shadow on hover
- Severity badge in top-right
- Clear action buttons at bottom

### Buttons
- Large, rounded
- Clear icons + text
- Hover effects
- Disabled states

### Layout
- Max width container
- Responsive (mobile-friendly)
- Generous spacing
- Cards in grid on desktop

## Accessibility
- High contrast text
- Clear focus states
- Keyboard navigation
- Screen reader friendly
